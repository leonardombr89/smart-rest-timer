Política de Privacidade — Smart Rest Timer

O Smart Rest Timer não coleta, armazena nem compartilha dados pessoais. O app funciona offline e não exige conta.

Permissões

  • Vibração: usada para fornecer feedback tátil quando o tempo termina.
  • Text-to-Speech: usada para falar uma mensagem curta ao finalizar o tempo.
